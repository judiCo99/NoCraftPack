Nom du Data Pack : NoCraftPack -

Version : Minecraft 1.21.4

Description :
Ce data pack désactive tous les crafts du jeu, sauf ceux du Dropper et du Crafter, qui ne peuvent être fabriqués qu’une seule fois.

Installation :

Activez les commandes lors de la création du monde pour que le pack fonctionne correctement.
Commandes utiles :

/reload – Si le pack ne se lance pas correctement.
/function nocraft:restart – Pour redémarrer le data pack.
/function nocraft:easymode – Active un mode plus facile.
Un défi Minecraft extrême où la fabrication est ultra-limitée.
